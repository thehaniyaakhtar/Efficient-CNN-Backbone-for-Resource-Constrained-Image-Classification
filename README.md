Efficient CNN Backbone with Modern Architecture

Built a lightweight CNN framework from scratch and compared multiple architectural improvements on image classification using CIFAR-10 image dataset.

Key Implementations
- Custom CNN Blocks (Conv → BN → ReLU)
- Residual Connections
- Squeeze-and-Excitation (SE) Attention
- MobileNet Depthwise Separable Convolutions
- Global Average Pooling
